Campo Harmônico de C Maior

O campo harmônico de C maior é formado pelos acordes: C, Dm, Em, F, G, Am, Bdim. Cada acorde pode ser tocado com arpejos, mostrando as notas da mão esquerda e direita.

Teclado Interativo

Acordes do Campo Harmônico

C maior
D menor
E menor
F maior
G maior
A menor
B diminuto

Exemplo de Música: C - Am - F - G
